Module Color

For playing with colors

Usage:

    local Color = require('stdlib/color/color')

Functions

Functions

set ([color=white][, alpha=1])
Return a color table with alpha added.

Parameters:

  • color defines.color or luaColor to set (default white)
  • alpha number the alpha to set on the color, or the existing alpha, or full alpha (default 1)

Returns:

    Color a color table with alpha added.
to_table (c_arr)
Converts a color in the array format to a color in the table format

Parameters:

  • c_arr array the color to convert

Returns:

    Color a converted color, { [r=carr[1]][, g=carr[2]][, b=carr[3]][, a=carr[4]] }
from_hex (hex[, alpha=1])
Return a color table with alpha added from a hexadecimal string.

Parameters:

  • hex string hexadecimal colour string (#ffffff, not #fff)
  • alpha number alpha value, from 0 to 1] (default 1)

Returns:

    Color
