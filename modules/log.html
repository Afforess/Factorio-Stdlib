Logger.new (mod_name, log_name, debug_mode)
Creates a new logger object

Parameters:

  • mod_name [required] the name of the mod to create the logger for
  • log_name (optional, default: 'main') the name of the logger
  • debug_mode (optional, default: false) toggles the debug state of logger. In debug mode, the logger writes immediately. Otherwise it buffers lines. Logger flushes after 60 seconds has elapsed since the last message.

Returns:

    the logger instance
Logger.log (msg)
Logs a message

Parameters:

  • msg a string, the message to log

Returns:

    true if the message was written, false if it was queued for a later write
Logger.write ()
Writes out all buffered messages immediately
