split (s, sep, pattern)
Splits a string into a table

Note: Empty split substrings are not included in the resulting table. For example, string.split('foo.bar...', '.', false) results in the table {'foo', 'bar'}

Parameters:

  • s the string to split
  • sep (optional) the separator to use. The period character, ., is the default separator.
  • pattern whether to interpret the separator as a lua pattern or plaintext for the string split

Returns:

    the table
