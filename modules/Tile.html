Module Tile

Tile module

A tile represents a 1x1 area on a surface in factorio

Functions

from_position (position) Calculates the tile coordinates for the position given
to_area (tile_pos) Converts a tile position to the area it contains
get_data (surface, tile_pos, default_value) Gets user data from the tile, stored in a mod's global data.
set_data (surface, tile_pos, data) Sets user data on the tile, stored in a mod's global data.
get_index (tile_pos) Calculates and returns a stable, deterministic integer id for the given tile_pos

Tile id will not change once calculated

Tile ids are only unique for the chunk they are in, they may repeat across a surface.

get_data (surface, tile_pos, default_value)
Gets user data from the tile, stored in a mod's global data.

The data will persist between loads

Parameters:

  • surface the surface to look up data for
  • tile_pos the tile coordinates to look up data for
  • default_value (optional) to set and return if no data exists

Returns:

    the data, or nil if no data exists for the chunk
set_data (surface, tile_pos, data)
Sets user data on the tile, stored in a mod's global data.

The data will persist between loads

Parameters:

  • surface the surface to look up data for
  • tile_pos the chunk coordinates to look up data for
  • data the data to set (or nil to erase the data for the tile)

Returns:

    the previous data associated with the tile, or nil if the tile had no previous data
