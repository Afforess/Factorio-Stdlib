Module Chunk

Chunk module

A chunk represents a 32x32 area of a surface in factorio.

Functions

from_position (position) Calculates the chunk coordinates for the tile position given
to_area (chunk_pos) Converts a chunk to the area it contains
get_data (surface, chunk_pos, default_value) Gets user data from the chunk, stored in a mod's global data.
set_data (surface, chunk_pos, data) Sets user data on the chunk, stored in a mod's global data.
get_index (surface, chunk_pos) Calculates and returns a stable, deterministic, unique integer id for the given chunk_pos

The id will not change once calculated


Functions

from_position (position)
Calculates the chunk coordinates for the tile position given

Parameters:

  • position to calculate the chunk for

Returns:

    the chunk position as a table

Usage:

    local chunk_x = Chunk.from_position(pos).x
to_area (chunk_pos)
Converts a chunk to the area it contains

Parameters:

  • chunk_pos to convert to an area

Returns:

    area that chunk is valid for
get_data (surface, chunk_pos, default_value)
Gets user data from the chunk, stored in a mod's global data.

The data will persist between loads

Parameters:

  • surface the surface to look up data for
  • chunk_pos the chunk coordinates to look up data for
  • default_value (optional) to set and return if no data exists

Returns:

    the data, or nil if no data exists for the chunk
set_data (surface, chunk_pos, data)
Sets user data on the chunk, stored in a mod's global data.

The data will persist between loads

Parameters:

  • surface the surface to look up data for
  • chunk_pos the chunk coordinates to look up data for
  • data the data to set (or nil to erase the data for the chunk)

Returns:

    the previous data associated with the chunk, or nil if the chunk had no previous data
get_index (surface, chunk_pos)
Calculates and returns a stable, deterministic, unique integer id for the given chunk_pos

The id will not change once calculated

Parameters:

  • surface the chunk is on
  • chunk_pos of the chunk
