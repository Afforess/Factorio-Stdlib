Factorio Standard Library

The Factorio Standard Library aims to extend the Factorio Lua api with well-tested, documented, and easy-to-use functions to make modding easier.

Modules

Area For working with bounding boxes.
Chunk A chunk represents a 32x32 area of a surface in factorio.
Position For working with x, y coordinates.
Surface A "domain" of the world.
Tile A tile represents a 1x1 area on a surface in factorio.
Color For playing with colors
Config Config module
Core Core module
Data For working with raw data.
Recipe For working with raw recipes.
defines.color A defines module for retrieving colors by name Extends the Factorio defines table
defines.time A defines module for retrieving the number of ticks in 1 unit of time.

Inventory For working with inventories
Event Makes working with events in factorio a lot more simple.
Event.Gui Gui Event module

Makes monolithic Factorio gui events more manageable.
Event.Time Raise events at certain times throughout a surfaces day, This module uses the stdlib Event system See also event.time.lua

Trains Trains module

When this module is loaded, it registers new events in order to track trains as locomotives and carriages are moved around.
