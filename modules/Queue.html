Module Queue

A double queue.

Taken from "Programming in Lua" (http://www.lua.org/pil/11.4.html) Modified to not allow nil values, and returns nil if popleft or popright is used when the queue is empty.

Usage:

    local Queue = require('stdlib/queue')

Functions

new () Constructs a new Queue object.
push_left (queue, value) Push a new element to the front of the queue.
push_right (queue, value) Push a new element to the back of the queue.
pop_left (queue) Retrieve the element at the front of the queue.
pop_right (queue) Retrieve the element at the back of the queue.
is_empty (queue) Returns true if the given queue is empty.
