Functions

get_resources_at (surface, position)
Gets all resource entities at the specified position and surface. Adapted from YARM/resmon.lua -> findresourceat

Parameters:

  • surface string or Surface the surface of the position
  • position Position the position to check

Returns:

    {nil or LuaEntity,...} the resource entities, can be empty if there are no resources there
get_resource_patches_at (surface, position)
From the resources at the given surface and position, return all connected (horizontally, vertically and diagonally) resource entities. (For now, just ore patches. Problems arise when a single resource entity spans multiple tiles). (Note that the implementation is not stable: If a resource entity reference changes during the search, the old and the new version of the entity might be included)

Parameters:

  • surface Surface the surface of the position
  • position Position the position to check

Returns:

    {nil or "resource_type" = {LuaEntity,...}, ...} a map of all resource types to their connected resource entities, or an empty array if there are no resources there
get_resource_patch_at (surface, position, type)
From the resources at the given surface and position, return all connected (horizontally, vertically and diagonally) resource entities of the specified type. (For now, just ore patches. Problems arise when a single resource entity spans multiple tiles). (Note that the implementation is not stable: If a resource entity reference changes during the search, the old and the new version of the entity might be included)

Parameters:

  • surface Surface the surface of the position
  • position Position the position to check
  • type string the resource type, i.e. "iron-ore"

Returns:

    {nil or LuaEntity,...} an array containing all resources in the resource patch, or an empty array if there are no resources there
get_resource_types (resources)
Given a list of resource entities, return a list of the different resource names.

Parameters:

  • resources {LuaEntity,...} a list of resource entities

Returns:

    {nil or string,...} a new list containing the names of the resources, can be empty
filter_resources (resources, resource_names)
Given a list of resource entities, return all resource entities with the given resource name.

Parameters:

  • resources {LuaEntity,...} a list of ore entities
  • resource_names {string,...} the name of the ore

Returns:

    {nil or LuaEntity,...} a new list containing the ores matching the given resource names, can be empty
get_resource_patch_bounds (resource_patch)
Given a resource patch, return the bounding box of the resource patch.

Parameters:

  • resource_patch {LuaEntity,...} the resource patch

Returns:

    BoundingBox the bounding box of the resource patch
