Entity.get_data (entity)
Gets user data from the entity, stored in a mod's global data.

The data will persist between loads, and will be removed for an entity when it becomes invalid

Parameters:

  • entity the entity to look up data for

Returns:

    the data, or nil if no data exists for the entity
Entity.set_data (entity, data)
Sets user data on the entity, stored in a mod's global data.

The data will persist between loads, and will be removed for an entity when it becomes invalid

Parameters:

  • entity the entity to set data for
  • data the data to set, or nil to delete the data associated with the entity

Returns:

    the previous data associated with the entity, or nil if the entity had no previous data
