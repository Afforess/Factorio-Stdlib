copy_as_simple_stacks (src, dest, clear)
Copies an inventory contents to a destination inventory using simple item stacks.

Parameters:

  • src source inventory to copy from
  • dest destination inventory, to copy to
  • clear clear the contents of the source inventory

Returns:

    an array of SimpleItemStacks of left over items that could not be inserted into dest.
each (inventory, func[, ...])
Given a function, apply it to each slot in the inventory. Passes the slot index as second argument to the function.

Iteration is aborted if the applied function returns true for any element during iteration.

Parameters:

  • inventory to be iterated
  • func to apply to values
  • ... additional arguments passed to the function (optional)

Returns:

    the slot index found or nil
