copy_as_simple_stacks (src, dest[, clear=false])
Copies an inventory contents to a destination inventory using simple item stacks.

Parameters:

  • src Inventory source inventory to copy from
  • dest Inventory destination inventory, to copy to
  • clear boolean clear the contents of the source inventory (default false)

Returns:

    {LuaSimpleItemStack,...} an array of left over items that could not be inserted into dest.
each (inventory, func[, ...])
Given a function, apply it to each slot in the inventory. Passes the slot index as second argument to the function.

Iteration is aborted if the applied function returns true for any element during iteration.

Parameters:

  • inventory Inventory to be iterated
  • func function to apply to values
  • ... additional arguments passed to the function (optional)

Returns:

    optional ItemStack the slot interation was aborted at or nil
each_reverse (inventory, func[, ...])
Given a function, apply it to each slot in the inventory. Passes the slot index as second argument to the function.

Iteration is aborted if the applied function returns true for any element during iteration. Iteration is performed from last to first in order to support dynamically sized inventories.

Parameters:

  • inventory Inventory to be iterated
  • func function to apply to values
  • ... additional arguments passed to the function (optional)

Returns:

    optional ItemStack the slot iteration was aborted at or nil
