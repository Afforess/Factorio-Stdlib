new (config_table)
Creates a new Config object to ease the management of a config table.

Parameters:

  • config_table [required] The table to be managed.

Returns:

    the Config instance for managing config_table

Usage:

  • --[Use a global table for config that persists across game save/loads]
ONFIG = Config.new(global.testtable)
  • --[You can also create a temporary scratch pad config]
ONFIG = Config.new({}) -- Temporary scratch pad
  • --[Setting data in Config]
ONFIG = Config.new(global.testtable)
ONFIG.set("your.path.here", "myvalue")
  • --[Getting data out of Config]
ONFIG = Config.new(global.testtable)
y_data = CONFIG.get("your.path.here")
  • --[Getting data out of Config with a default to use if path is not found in Config]
ONFIG = Config.new(global.testtable)
y_data = CONFIG.get("your.path.here", "Your Default here")
  • --[Deleting a path from Config]
ONFIG = Config.new(global.testtable)
ONFIG.delete("your.path.here")
  • --[Checking if a path exists in Config]
ONFIG = Config.new(global.testtable)
ONFIG.is_set("your.path.here")
get (path, default)
Get a stored config value.

Parameters:

  • path [required] a string representing the variable to retrieve
  • default (optional) value to be used if path is nil

Returns:

    value at path or nil if not found and no default given
set (path, data)
Set a stored config value.

Parameters:

  • path [required] a string, config path to set
  • data (optional) Value to set path to. If nil it behaves identical to Config.delete()

Returns:

    number 0 on failure; number of affected paths on success
delete (path)
Delete a stored config value.

Parameters:

  • path a string, config path to delete

Returns:

    number 0 on failure; number of affected paths on success
is_set (path)
Test the existence of a stored config value.

Parameters:

  • path a string, config path to test

Returns:

    boolean, true on success, false otherwise
